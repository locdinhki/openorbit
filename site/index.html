❯ Quick Start

All platforms releases
# Clone, install, and run. That's it.
$ git clone https://github.com/locdinhki/openorbit.git && cd openorbit
$ npm install && npm run dev
# Download the latest .dmg and open it
$ curl -fsSL https://github.com/locdinhki/openorbit/releases/latest/download/openorbit-1.0.0.dmg -o OpenOrbit.dmg
$ open OpenOrbit.dmg
# Download the installer from GitHub Releases
> Invoke-WebRequest https://github.com/locdinhki/openorbit/releases/latest -OutFile openorbit-setup.exe
> .\openorbit-setup.exe
# Download the AppImage and make it executable
$ wget https://github.com/locdinhki/openorbit/releases/latest/download/openorbit-1.0.0.AppImage
$ chmod +x openorbit-1.0.0.AppImage && ./openorbit-1.0.0.AppImage

Works on macOS, Windows & Linux. Build from source or grab a binary.

Land Your Next Role

Multi-AI Scoring

4 AI providers (Claude SDK, Claude API, OpenAI, Ollama) analyze every listing, score match quality 0–100, and flag red flags.

Easy Apply Engine

Full LinkedIn Easy Apply automation with multi-step form detection, answer templates, and intelligent question handling.

Stealth Mode

Human behavior simulation with random delays, typing variation, idle pauses, and Patchright anti-detection built into every action.

Extension System

9 extensions with hot-loading, shared services, and isolated IPC. Add AI providers, job platforms, or messaging bots as plugins.

Remote Control

Manage your job search from Telegram, iMessage, or WhatsApp. AI-powered bots with inline keyboards and direct commands.

Database Tooling

Built-in schema browser, data grid, SQL console, and export/import. Full visibility into your job search data.

How OpenOrbit Works

From search to application in three simple steps.

1

Configure Your Profile

Set up your skills, preferences, target roles, and salary expectations. OpenOrbit builds your candidate profile.

2

AI Scans & Scores

The engine continuously discovers opportunities across platforms and uses Claude AI to rank every match.

3

Auto-Apply

Top matches get applied to automatically. You review, approve, and track everything from any device.
